TROUBLESHOOTING:
────────────────

Issue: "Table shows 0 applications"
→ Check if you have data in localStorage
  Dev Console: Dashboard.StorageManager.loadConfig().apps
  → Should show array with applications
  → If empty: Add test data via Applications tab first

Issue: "KPI cards show undefined or 0"
→ Check if data is loaded correctly
  Dev Console: Dashboard.StorageManager.getAllApps()
  → Should return list of applications

Issue: "Buttons do nothing when clicked"
→ Check console for errors (F12 → Console)
→ Look for: calculateAppProgress, renderAppsOverviewTable, etc.
→ Report any red errors

Issue: "Table is empty even with data"
→ Check filter status (click "All" to reset)
→ Check search box is empty
→ Try page refresh (Ctrl+R)

Issue: "Animations are stuttering"
→ Normal - depends on browser performance
→ Try restarting browser or using different browser

Issue: "After making changes, data doesn't update"
→ Click Applications Overview tab again
→ Or manually trigger: Dashboard.AppsOverviewWorldClass.refresh()
