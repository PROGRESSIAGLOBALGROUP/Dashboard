CÓMO USAR LOS NUEVOS CAMPOS

EN LA TABLA:

1. Editar Order:
   • Haz clic en la celda del número
   • Cambia el valor (mín 1)
   • Presiona Enter o Tab para guardar
   ✅ Se guardará automáticamente

2. Editar Wave:
   • Haz clic en el dropdown de Wave
   • Selecciona Wave 1, 2, o 3
   • Se guardará automáticamente

EN EL FORMULARIO (Crear nueva aplicación):

1. Ingresa todos los datos de la aplicación
2. Asigna un Order (default: 1)
3. Selecciona una Wave (default: Wave 1)
4. Presiona "Create Application"
✅ Se guardará en la estructura de datos
