🚀 CÓMO USAR:

   CREAR APLICACIÓN CON ORDER Y WAVE:
   ├─ Ve a Project Administration > Applications
   ├─ Selecciona Business Unit
   ├─ Click "Add Application"
   ├─ Completa el formulario
   ├─ Establece Order (ej: 2)
   ├─ Selecciona Wave (ej: Wave 2)
   └─ Click "Create Application"

   EDITAR EN LÍNEA:
   ├─ Haz clic en la celda Order o Wave
   ├─ Cambia el valor
   ├─ Presiona Enter/Tab
   └─ ✅ Se guarda automáticamente
